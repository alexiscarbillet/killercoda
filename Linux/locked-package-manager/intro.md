# Troubleshooting Package Manager Locks

In this scenario, you are a System Administrator who has just logged into a server to install some essential tools. However, you'll find that the package manager (`apt`) is unresponsive and throwing errors.

### What you will learn:
* How to interpret `dpkg` and `apt` lock errors.
* How to use `lsof` and `ps` to find blocking processes.
* How to safely clear stale locks and recover the package database.
