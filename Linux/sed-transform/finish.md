### Key Takeaways:

* **Substitution:** `sed 's/pattern/replacement/' file`
* **Deletion:** `sed '/pattern/d' file`
* **In-Place Editing:** You **must** use the **`-i`** flag to save changes back to the file. Without it, `sed` only prints to the screen.
